# QRventory

QRventory es una aplicación para la **gestión de inventarios mediante códigos QR y de barras**, pensada para facilitar el control de productos con terminales y su registro en **Excel y Google Sheets** de forma rápida y organizada.

La aplicación permite registrar, consultar y actualizar información de inventario escaneando códigos QR, reduciendo errores manuales y optimizando el tiempo de gestión.

---

## Características principales

- Gestión de inventario leyendo **códigos QR y de barras**
- Exportación y manejo de datos en **Excel y Google Sheets**
- Registro y consulta rápida de productos
- Interfaz sencilla y fácil de usar

---

## Tecnologías utilizadas

- Lenguaje principal: **KOTLIN**
- Manejo de archivos Excel
- Google Sheets API
- Generación y lectura de códigos QR y de Barras
